# 网页内容摘要工具 - 规格说明

## 1. 需求概述

创建一个命令行工具，输入网页 URL 后自动抓取正文内容，使用本地 AI 模型生成 100-300 字的摘要。

## 2. 文件结构

```
web-summarizer/
├── config.example.yaml    # 配置模板
├── config.yaml            # 用户配置（手动创建）
├── requirements.txt       # Python 依赖
└── web_summarizer.py      # 主程序
```

## 3. 配置参数

| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| model_type | 模型类型 (llama/qwen/chatglm) | qwen |
| model_path | 模型文件路径 | ./models/ |
| max_tokens | 摘要最大 token 数 | 300 |
| timeout | 网页请求超时（秒） | 30 |

## 4. 核心功能

1. 命令行参数解析（URL 输入）
2. 网页抓取与正文提取
3. 本地模型加载与推理
4. 摘要生成与格式化输出

## 5. 输出格式

```
========================================
网页内容摘要
========================================
URL: xxx
标题: xxx
========================================
摘要:
xxx (100-300字)
========================================
耗时: x.xs
```
