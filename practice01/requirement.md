# 网页内容摘要工具 - 需求说明

## 一、用户目标

作为开发者，我需要一个命令行工具，能够：
- 输入任意网页 URL，自动抓取网页正文内容
- 使用本地部署的 AI 模型生成 100-300 字的摘要
- 支持多种主流本地模型切换
- 提供简洁清晰的命令行输出

## 二、功能需求

### 2.1 网页抓取
- 支持 HTTP/HTTPS 协议的网页 URL 输入
- 自动识别并提取网页正文内容（去除广告、导航、页脚等无关内容）
- 处理常见网页编码问题（UTF-8、GBK 等）
- 支持超时设置，避免长时间等待

### 2.2 摘要生成
- 使用本地模型生成网页内容摘要
- 摘要长度控制在 100-300 字
- 支持 Llama、Qwen、ChatGLM 等多种本地模型
- 提供模型配置选项

### 2.3 命令行交互
- 简洁的命令行参数输入
- 清晰的进度提示
- 友好的错误信息提示
- 输出格式规范易读

### 2.4 配置管理
- 提供配置文件，支持自定义模型路径、参数等
- 支持命令行参数覆盖配置文件
- 配置模板文件示例

## 三、使用流程

1. 安装依赖和下载本地模型
2. 编辑配置文件，设置模型路径和参数
3. 运行命令：`python web_summarizer.py --url <网页URL>`
4. 查看生成的摘要

## 四、输入输出示例

### 输入
命令行输入：
```
python web_summarizer.py --url https://example.com/article
```

### 输出
程序执行后的控制台输出：
```
========================================
网页内容摘要
========================================
URL: https://example.com/article
标题: 示例文章标题
========================================
摘要:
这是生成的 100-300 字摘要内容，准确概括了网页的主要信息。
========================================
耗时: 5.2s
```

## 五、约束条件

- 使用 Python 开发
- 使用本地模型，不依赖云端 API
- 支持多种主流本地模型
- 命令行工具，无图形界面
- 纯核心功能，不包含批量处理、历史记录等额外功能
