Summary of tool_client.py:
This Python script defines a set of utility tools for interacting with the local filesystem and web, designed to be used by an LLM agent. It includes:
1. Tool functions:
   - list_files(directory): lists files and directories with metadata.
   - read_file(file_path): reads file content.
   - create_file(file_path, content): creates a file, making directories as needed.
   - fetch_web_content(url): retrieves webpage HTML.
2. A ChainedCallContext class that tracks steps, variables, and iteration limits for multi-step tool calls.
3. A call_llm function that wraps the OpenAI-compatible API request, handling tool usage.
4. AVAILABLE_TOOLS dictionary mapping tool names to their implementations.
5. TOOLS_SCHEMA list describing each tool's JSON schema for function calling.
6. Loading of environment variables (.env) for API key, base URL, and model configuration.
The file contains numerous 'def' statements defining these functions and classes.
